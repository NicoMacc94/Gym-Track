# Gym Track – web

Scheletro Next.js + React + TypeScript creato con `create-next-app`.

## Requisiti
- Node.js 18.18+ (consigliato l’ultimo LTS)
- npm (incluso con Node)

## Setup rapido
```bash
cd web
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) per vedere l’app di esempio.
